Git Clone e Git Pull

Git Clone

Serve para clonar um repositório já existente para o PC que estamos trabalhando

Passos para criar um Git Clone:

  1. Dar um git bash here na pasta que o clone será criado;

  2. Digitar o comando git clone e a url do repositório que será clonado. Exemplo: git clone https://github.com/RogerioCosta88/rogeriocosta88.git e teclar o Enter;

Pronto o respositório foi clonado corretamente.

Git Pull

O git pull serve para atualizar um projeto já existente na nossa máquina, mas esta desatualizado.

O git pull pegará as informações novas do GitHub e as atualizará no nosso projeto local, sem alterar as demais informações.

Passos para criar um Git Pull:

  1. Na pasta do projeto linkado, dar um git pull e teclar Enter.

Pronto, o projeto já está atualizado.

Obs.: as vezes é necessário especificar o nome do branch, exemplo: git pull main
